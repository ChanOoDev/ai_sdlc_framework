# Doctor Note Domain Skill

Use this skill for clinic note-taking applications.

## Core Workflows

- Doctor registration
- Patient registration
- Patient profile
- Consultation notes
- Patient history
- Search
- PDF export

## Domain Rules

- Protect patient data.
- Restrict consultation creation to doctors/admins.
- Keep receptionist access limited to patient demographics.
- Avoid billing, appointment, pharmacy, lab, and insurance in MVP unless approved.
