# Third-Party Recommendations

You can install third-party agents, skills, MCPs, commands, and hooks from community marketplaces such as AITMPL or Claude Code template repositories.

## Recommended Third-Party Agents

- Frontend Developer
- Backend Developer
- Database Architect
- Code Reviewer
- Security Auditor
- DevOps Engineer
- Technical Writer
- Debugger
- Performance Optimizer

## Keep Custom

- Orchestrator
- Product Manager
- Scrum Master
- Release Manager
- Domain skill
- AI SDLC skill

## Rule

Community agents provide technical execution. Custom agents preserve your delivery process.
